# Banreservas QA Automation Portfolio (API + UI)

Framework de automatización de pruebas QA utilizando **Playwright Test + TypeScript**, que incluye:

- Automatización de API Testing (CRUD)
- Automatización UI Testing con Page Object Model (POM)
- Reportes HTML de Playwright
- Configuración por entorno (.env)

Este proyecto demuestra habilidades de **SDET / QA Automation Engineer**.

---

# Tecnologías utilizadas

- Playwright Test
- TypeScript
- Node.js
- REST API Testing
- Page Object Model (POM)
- dotenv
- Git

---

# Instalación del proyecto

## 1. Clonar el repositorio

```bash
git clone https://github.com/J0224/Banreservras-API-Automation.git
cd Banreservras-API-Automation
```
# Instalacion de dependecias

```bash
cd Banreservras-API-Automation
npm install
```

# correr proyecto

```bash
cd Banreservras-API-Automation
npx playwright test
```
